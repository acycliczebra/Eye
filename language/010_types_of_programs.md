
# Types of Programs #

1. Apps
2. Scripts
3. Rule
4. Libs


| Type          | DB Access                 | IO    |
| ------------- |:-------------------------:|:-----:|
| App           | read + write (continuous) | yes   |
| Script        | write only                | yes   |
| Rule          | read + write              | no    |
| Libs          | n/a                       | n/a   |
